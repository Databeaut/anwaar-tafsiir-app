# Anwaar Tafsiir App

A premium, interactive Quran Tafsiir application focusing on Surah Al-Fatiha.

## Features

- **Smart Video Player**: Seamless Youtube integration with custom segmentation and controls.
- **Glassmorphism UI**: High-end, sleek design with smooth animations.
- **Interactive Lessons**: Automated lesson tracking, unlocking, and completion logic.
- **Ayah Synchronization**: Real-time Ayah display synced with video playback.
- **Student Management**: Admin dashboard for generating branded access keys (`ANWAAR` + 4 digits).
- **Authentication**: Secure student login system.

## Tech Stack

- **Frontend**: React, TypeScript, Tailwind CSS, Framer Motion.
- **Backend/Database**: Supabase (Auth & Postgres).
- **State Management**: React Context & Hooks.

## Getting Started

1.  Clone the repository.
2.  Install dependencies: `npm install`
3.  Run development server: `npm run dev`

## Deployment

Configured for Vercel deployment with SPA routing support.
